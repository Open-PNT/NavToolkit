Units
=====

What Units Does NavToolkit Use?
-------------------------------

NavToolkit uses SI units, unless explicitly noted.
Here is a non-exhaustive list of units used in NavToolkit for various quantities:

- Distance: meters
- Velocity: :math:`\frac{\text{meters}}{\text{seconds}}`
- Acceleration: :math:`\frac{\text{meters}}{\text{seconds}^2}`
- Angles: radians
